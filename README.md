# Harvard Innovation Labs — Summer 2026 Programming Reference

An interactive HTML reference document for the HIL Summer 2026 programming system. Provides a structured, at-a-glance overview of all five programming containers, their delivery mechanisms, content, entry models, and target audiences.

---

## Overview

This single-file HTML artifact maps the full HIL summer programming landscape across five containers:

| Container | Focus | Entry Model |
|---|---|---|
| 🌱 Founder Fundamentals | Idea validation, customer discovery | Intentional activation |
| 🏛️ Venture Incubation | Milestone-gated progression | Intentional activation + selection |
| 🚀 Acceleration | Cohort-based company building | Intentional activation + selection |
| 📈 Venture Growth | Expert talks, open workshops | Minimal activation |
| 🤝 Open Community | Social events, peer circles | Minimal activation |

---

## Features

- **Expandable cards** — click any container to reveal its delivery mechanisms, content, and entry flow
- **Color-coded themes** — each container has a distinct color for quick visual scanning
- **Entry model legend** — explains the three activation levels used across programming
- **Entry flow diagrams** — step-by-step paths showing how founders enter each container
- **Responsive layout** — adapts to mobile and desktop viewports

---

## Usage

Open `hil_summer_2026_programming.html` directly in any modern browser. No server, build step, or dependencies required beyond a Google Fonts network request for Inter and Playfair Display.

---

## Structure

```
hil_summer_2026_programming.html
├── Header — title, dates, location, program count
├── Logic bar — framework labels (Entry / Container / Content / Delivery)
├── Container cards (×5, accordion)
│   ├── Delivery mechanisms
│   ├── Content this summer
│   ├── Who it's for
│   └── Entry flow
├── Entry model legend
└── Footer — notes on unconfirmed dates
```
